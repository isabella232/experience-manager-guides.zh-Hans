# 查看翻译状态 {#id169SEK00KOW}

您可以在DITA映射中查看每个主题的翻译状态和翻译的语言副本。

执行以下步骤以查看DITA映射的翻译状态：

1. 导航到源语言副本的DITA映射文件。
1. 单击 **翻译** 选项卡。
1. 在 **筛选** 面板中，选择 **翻译语言** 要检查其状态并单击 **完成**.

   >[!NOTE]
   >
   > 您可以根据内容的翻译状态\（如不同步、缺少副本、进行中或同步\）、源类型\（如全部、DITA、DITA映射或资源\）、修改日期进一步筛选内容。 您还可以输入关键字以搜索特定主题。
